杂牌网卡使用猎豹开wifi怎么都连不上，蛋疼 (> _< !!)

记一下手动开启wifi的过程

使用netsh命令


netsh wlan

hostednetwork mode=allow ssid=tmpwifi key=wifi0x0000 keyUsage=temporary

hostednetwork start


写成批处理:

@echo off
netsh wlan
hostednetwork mode=allow ssid=tmpwifi key=wifi0x0000 keyUsage=temporary
hostednetwork start

之后调整共享的网络适配器就行了

不过很好奇有没有设置网络分享的时候有没有相应的命令啊？

这种方式开启的wifi有时候会连接不上，试多几次可以了，就算是这样的方式应该也有优化的地方的。
