使用crontab

有两个文件规定了user的使用权限(默认每个用户拥有自己的crontab文件)

/etc/cron.allow: If this file exists, only users listed in it can use crontab.

/etc/cron.deny: If this file exists, users listed in it cannot use crontab.


全部user的crontab文件存放在/var/spool/cron/crontabs/中，一般的user无法编辑这里的文件，需要使用crontab命令来操作。


分类目录存放特定时间的脚本:

/etc/cron.d
/etc/cron.daily
/etc/cron.hourly
/etc/cron.monthly
/etc/cron.weekly

具体执行时间各个系统有所不同，也可以在/etc/crontab中调整。


crontab文件格式


* * * * * Command to be executed
- - - - -
| | | | |
| | | | +----- Day of week (0-6)
| | | +------- Month (1 - 12)
| | +--------- Day of month (1 - 31)
| +----------- Hour (0 - 23)
+------------- Min (0 - 59)
