不知道为什么就是想手动来写这个博客，虽然有很多的框架，之前也是有用过bootstrap，但是莫名的想要一种简便的风格，纯粹的记录内容而已。

了解了一下html5中的新元素，决定将aside放置在右边并且用虚点线做必要的分隔。

需要几个css样式:

1.左浮动

.right_sidebar{width:20%; float:right;}

2.左边框虚线

.border_left_dashed{ border:1px; border-style:none none none dashed; }

3.底边框虚线

.border_left_dashed{ border:1px; border-style:none none dashed none; }

header aside main 以及 footer元素对应套上虚线样式即可...
