if __name__ == '__main__'作用为判断是否为主文件执行方式。

  1. 一个python文件有两种使用方式：直接当作脚本执行 和 当作模块导入。

  2. 当直接当作脚本执行时候，__name__的值为__main__

  3. 当作为模块导入时，__name__的值为模块名称。

  4. 固if __name__ == '__main__'下的代码只有当这个文件被当作脚本文件执行时候才会执行。
