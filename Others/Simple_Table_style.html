在HTML中简洁地使用Table

  1. 默认的table有边距，有时看上去很冗余，设置了border之后它们一般是这个样子的：

    BeiJin HongKong
    ShangHai GuangZhou

  2. 其实只要简单的几句css便可让table简介，无边距的table有时其实更加适合阅读：

    CSS代码：
				
        table {
                    border-collapse: collapse;
                }

        table, td, th {
                    border: 1px solid black;
        }
                
            
    一个简介的Table看起来是这样的：
                
    
                    
    
                        BeiJin
                        HongKong
                    
    
                    
    
                        ShangHai
                        GuangZhou
