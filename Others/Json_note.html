json经常用户前后台的数据交换，ajax中经常使用的数据封装格式是xml和json，xml解析会比较麻烦，通常情况下尽量使用的json。

  1. json格式包含基本的键值对: "key":"word"，他们可以类比对象中的变量名和变量值，可以使用英文逗号分隔开，用大括号{}将多个键值对括起，就代表了一个对象；多个对象也可以组成数组，他们使用中括号[]括起并用逗号分隔。

  2. 调用方式: array[index].key[...];也可以是array[index][key][...]方式。
    例如: [{"City":"Beijing","ZipCode":"111111"},{"City":"Langfang","ZipCode":"222222"}]
    可以这样调用:arrayName[0].city
    也可以这样调用:arrayName[0][city]

  3. 对于key为一个数字字符的情况，不可以使用"."来作为连接符，只能使用[]并且还可以加上单引号对''。
    例如: [{"City":"Beijing",1:"111111"},{"City":"Langfang",2:"222222"}]
    应该这样调用: 

    				arrayName[0][1]
				或者:
				arrayName[0]['1']
