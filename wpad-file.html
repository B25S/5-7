Title:LAN中获取wpad.dat文件 Keywords: Catagory: Description: Author:B25S license: -----

wpad全称为Web Proxy Autodiscovery Protocol，用户LAN中自动设置web代理。

  1. 确定代理服务的存在，可以直接使用 ping wpad

  2. ping过之后得到了代理服务器的ip地址，打开浏览器输入 http://ip/wpad.dat,即可下载到wpad.dat文件。

  3. 在浏览器中设置自动的检测web代理的时候，其实使用的就是这份文件，浏览器利用这个协议自动的获得代理服务器的ip等各种信息。

  4. 可以使用netstat命令来确定代理服务器地址。

  5. ifconfig/all可以查看到primary dns suffix，使用ping wpad.primary dns suffix可以检测代理服务器地址。

后记

  1. 如果服务器启动了dhcp，ping wpad.primary dns suffix可能不适用。

  2. wpad.dat实际上应该是一份js文件。

  3. firefox设置自动检测web代理时，存在不定时的验证要求，版本50.1.0。
