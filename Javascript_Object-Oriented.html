javascript面向对象，参考:http://www.ruanyifeng.com/blog/2010/05/object-oriented_javascript_encapsulation.html

  1. javascript中没有类，也没有对象，javascript中使用 var o={A:'a',B:'b'} (json)来代表一个集合，类似对象模板-->类。

  2. 也可以使用方法来达成,function o(arg1,arg2){return {arg1:'one',arg2:'two'}}

    但是不能实现复用这样的面向对象特征。

  3. javascript中提供了this和new关键字，可以在方法中使用(>_<...只能说真很奇葩...,对习惯java的部分人来说简直就是不伦不类...)。
    this用于方法中，new用于对象申明:var o = new object();
    也就是说申明一个方法就是申明一个对象... o_O!!

  4. this和new的使用依然没有办法实现复用，即使拥有相同的属性，却不能像java一类那样抽象出来一个父类或者接口。(这里只是类比，应该是不准确)但是javascript中提供了prototype，它用于定义相同的共用的属性的值，指定值之后，所有new出来的变量'var'都是指向这个值的。

  5. javascript提供了isPrototypeOf()，hasOwnProperty()方法和in晕算法，可以验证对象与实例之间的关系。
